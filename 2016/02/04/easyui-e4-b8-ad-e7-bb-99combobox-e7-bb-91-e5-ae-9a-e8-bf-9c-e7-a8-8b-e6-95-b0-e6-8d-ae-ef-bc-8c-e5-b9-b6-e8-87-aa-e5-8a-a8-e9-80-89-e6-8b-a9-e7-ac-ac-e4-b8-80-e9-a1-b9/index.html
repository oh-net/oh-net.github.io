EASYUI中给combobox绑定远程数据，并自动选择第一项

Posted on 二月 4, 2016 本文总阅读量次

easyui中使用远程数据给combobox绑定值，然后自动选择第一项（同时禁用编辑）

$(“#c_source”).combobox({
url:’async/parameter_search.ashx?c=csource’,
method:’get’,
valueField: ‘KeyName’,
textField: ‘KeyValue’,
editable: false,
panelHeight: ‘auto’,
onLoadSuccess: function () {
console.log(“c_source load success”);
var val = $(this).combobox(“getData”);
for (var item in val[0]) {
if (item == “KeyName”) {
$(this).combobox(“select”, val[0][item]);
}
}
}
});
