通过SQL累加数据

Posted on 一月 23, 2018 本文总阅读量次

需求

通过SQL累加数据。已知有列A（id），B（coin），C（direction）， 规则： 1. 当 c = 1 时，查询的结果， 新列D == B（coin），否则 新列E == B（coin） 2. 然后再根据C（direction）的值，使用相同的规则(#1)，累加B（coin），生成新列F 先看表结构：

CREATE TABLE `test1` (
  `id` int(10) unsigned NOT NULL AUTO_INCREMENT,
  `coin` int(11) NOT NULL,
  `direction` int(11) NOT NULL,
  PRIMARY KEY (`id`)
);

--- 要实现的效果： : 也就是说：

  1. D=增加=B
  2. E=减少=B
  3. F=余额累加数

----

方案一

SET @csum := 0;
SELECT
    a.id,
    a.coin,
    a.direction AS 'B(direction)',
    (
        CASE a.direction
        WHEN 1 THEN
            a.coin
        ELSE
            ' '
        END
    ) AS D,
    (
        CASE a.direction
        WHEN 0 THEN
            a.coin
        ELSE
            ' '
        END
    ) AS 'E',
    (
        @csum := @csum + (
            a.coin * (
                CASE a.direction
                WHEN 1 THEN
                    1
                ELSE
                    - 1
                END
            )
        )
    ) AS 'F'
FROM
    test1 a
GROUP BY
    a.id,
    a.coin,
    a.direction,
    D,
    E

方案二

SELECT
    a.id,
    a.coin,
    a.direction AS 'B(direction)',
    (
        CASE a.direction
        WHEN 1 THEN
            a.coin
        ELSE
            ' '
        END
    ) AS D,
    (
        CASE a.direction
        WHEN 0 THEN
            a.coin
        ELSE
            ' '
        END
    ) AS 'E',
    (
        ifnull(
            (
                SELECT
                    SUM(b.coin)
                FROM
                    test1 b
                WHERE
                    b.id <= a.id
                AND b.direction = 1
            ),
            0
        ) - ifnull(
            (
                SELECT
                    SUM(b.coin)
                FROM
                    test1 b
                WHERE
                    b.id <= a.id
                AND b.direction = 0
            ),
            0
        )
    ) axxx
FROM
    test1 a
GROUP BY
    a.id,
    a.coin,
    a.direction,
    D,
    E
